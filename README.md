Have you ever wanted to make a Minecraft bed texture resource pack because it sounds fun?



1.  Run main.py
    - Using Python
        -   Needs [Python](https://www.python.org/downloads/) (Uses 3.13)
        -   Needs [Pillow](https://pillow.readthedocs.io/en/stable/installation/basic-installation.html) (Uses 12.1.1)
        -   Needs [Streamlit](https://docs.streamlit.io/get-started/installation) (Uses 1.54.0)
        -   `streamlit run main.py`
    -   Using [uv](https://docs.astral.sh/uv/getting-started/installation/) would be easier, but not required
        -   `uv run streamlit run main.py`
    
2.  Generate your new bed textures
    -   Select the bed color
    -   Select your custom image
    -   Download your modified bed texture

3.  Copy modified beds to the `textures/entity/bed` directory in your resource pack folder
    -   [How to make a Bedrock texture pack](https://learn.microsoft.com/en-us/minecraft/creator/documents/resourcepack?view=minecraft-bedrock-stable)
    -   [Vanilla Bedrock resource pack reference](https://github.com/Mojang/bedrock-samples/tree/main/resource_pack)
    -   Where you will end up putting your textures
        -   `%appdata%/Minecraft Bedrock/users/shared/games/com.mojang/development_resource_packs`
        -   Create your resource pack folder in above directory

4. View your new Minecraft beds in-game
